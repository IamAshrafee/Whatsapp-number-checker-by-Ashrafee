## The Problem

For marketing teams and individuals who use WhatsApp to connect with their audience, verifying which phone numbers are active on the platform is a tedious and time-consuming task. The manual process of copying and pasting each number from a list into WhatsApp to check for a valid profile is slow, inefficient, and prone to errors. This repetitive work takes away valuable time that could be spent on more important marketing activities.

## The Solution

This browser extension streamlines the process of verifying WhatsApp numbers directly within the WhatsApp Web interface. It helps you quickly identify which numbers from a list are active on WhatsApp, saving you hours of manual work. The extension is designed to mimic human-like behavior, with random delays and pauses, to ensure a smooth and reliable checking process. It's not a bulk messaging tool, but a utility to assist in validating your contact lists efficiently.

## How It Works

1.  **Paste Your Numbers:** You paste a list of numbers into any WhatsApp chat and send it as a message.
2.  **Open the Extension:** Click on the extension's icon in your browser's toolbar.
3.  **Start Checking:** Click the "Start Checking" button.
4.  **Automatic Scanning:** The extension automatically reads the last message, extracts the numbers, and checks each one.
5.  **View Results:** Valid numbers are displayed in the "Results" table in real-time.
6.  **Export Your Data:** Once the process is complete, you can export the results.

## Installation

Since the extension is not yet available on the Chrome Web Store, you need to install it manually using Developer Mode.

1.  **Download the Code:** Download the project files from this repository, preferably as a ZIP file from the "Releases" section.
2.  **Extract the Files:** Unzip the downloaded file into a folder on your computer.
3.  **Open Extensions Page:** Open your browser (Google Chrome, Microsoft Edge, etc.) and navigate to the extensions management page (e.g., `chrome://extensions`).
4.  **Enable Developer Mode:** Turn on the "Developer mode" toggle, which is usually in the top-right corner.
5.  **Load the Extension:** Click the "Load unpacked" button and select the folder where you extracted the project files.
6.  **Done!** The extension is now installed and ready to use.
